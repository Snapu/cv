# cv

## Project setup
```
npm install
```

### Configuration

1. Provide ```VUE_APP_NAME```, ```VUE_APP_ADDRESS```, ```VUE_APP_TEL```, ```VUE_APP_EMAIL```, ```VUE_APP_WEBSITE```in the environment. E.g.: Provide a .env.local file in the root directory of this project.

1. Edit json files in ```@/data```.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
